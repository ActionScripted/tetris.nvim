# tetris.nvim

Pure Lua tetris game for Neovim.

## Installation

Using lazy.nvim:

```lua
{
  "ActionScripted/tetris.nvim",
  dependencies = { "nvim-lua/plenary.nvim" },
  cmd = { "Tetris" },
  opts = {},
}
```
